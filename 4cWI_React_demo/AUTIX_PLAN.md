# Autix – Dein Auto. Deine Regeln.

Willkommen bei Autix, der innovativen Plattform, auf der du dein Traumauto ganz nach deinen Vorstellungen gestalten kannst. Egal ob sportlich, elegant oder praktisch – bei uns bestimmst du jedes Detail selbst.

## Was ist Autix?

Autix ist eine Website, auf der Nutzer ihr eigenes Auto individuell konfigurieren können.

## Ziel

Einfache, übersichtliche und personalisierte Auto-Konfiguration mit Preisberechnung.

## Zielgruppe

- Autoliebhaber
- Technik-Interessierte
- Junge Nutzer, die gerne Dinge selbst gestalten

## Funktionen der Website

- Auto konfigurieren (Farbe, Modell, Ausstattung)
- Motor auswählen
- Extras hinzufügen (z. B. Sitzheizung, Sound-System)
- Live-Preisberechnung
- Verschiedene Konfigurationen vergleichen

## USP / Besonderheiten

- Echtzeit Preisanzeige 💰
- Einfache Bedienung 👍
- Viele Individualisierungsmöglichkeiten 🎨
- Modernes Design

## Aufbau der Website

- Startseite
- Konfigurator
- Zusammenfassung
- Preisübersicht

## Design Ideen

- Moderne Farben (z. B. Schwarz, Weiß, Blau)
- Große Bilder von Autos
- Einfache Navigation
- Klare Buttons (z. B. „Jetzt konfigurieren“)

## Technologien

- HTML
- CSS
- JavaScript
- Datenbank für Speicherung
- Firebase / Supabase

## Zukunft / Erweiterungen

- Speichern von Konfigurationen
- Login-System
- Vergleich von Autos
- Direkt bestellen

## 4 Wochen Sprint Plan

### Sprint Ziel

In 4 Wochen entsteht ein funktionierender MVP von Autix, bei dem Nutzer:

- ein Auto konfigurieren können 🚘
- Optionen auswählen können 🎨
- den Preis sehen 💰
- eine einfache, saubere Web-App nutzen 🌐

### Woche 1 – Planung & Grundstruktur

Ziele:

- Grundlage für das gesamte Projekt schaffen

Aufgaben:

- Projektidee final definieren (Autix MVP)
- Zielgruppe festlegen
- Features priorisieren (Must / Should / Could)
- Wireframes (Startseite + Konfigurator)
- Tech Stack festlegen (z. B. React, Vue oder Vanilla JS)
- Repository erstellen (GitHub)
- Basis-Layout der Website bauen

Ergebnis:

- Grobes Design + funktionierende Seitenstruktur

### Woche 2 – UI & Konfigurator Aufbau

Ziele:

- Visuellen Konfigurator bauen

Aufgaben:

- Fahrzeugauswahl UI erstellen
- Design für:
  - Modellwahl
  - Farben
  - Felgen
  - Extras
- Interaktive Buttons / Auswahlfelder
- State Management vorbereiten (Speicherung der Auswahl)
- Erste Version des Konfigurators sichtbar machen

Ergebnis:

- Nutzer kann Optionen anklicken und sieht Änderungen

### Woche 3 – Preislogik & Funktionalität

Ziele:

- Konfigurator wird funktional & dynamisch

Aufgaben:

- Preisbasis definieren (Grundauto)
- Preislogik implementieren:
  - Farbe +X €
  - Felgen +X €
  - Extras +X €
- Live-Preisberechnung in Echtzeit
- Zusammenfassung der Konfiguration
- Datenstruktur für Auto-Optionen optimieren

Ergebnis:

- Voll funktionierender Konfigurator mit Preisberechnung

### Woche 4 – Testing, Design & MVP Release

Ziele:

- Produkt fertigstellen & verbessern

Aufgaben:

- UI/UX verbessern (Spacing, Farben, Fonts)
- Mobile Responsiveness
- Bugs fixen
- Performance optimieren
- Finale Tests (User Flow prüfen)
- Landing Page verbessern
- MVP Präsentation vorbereiten

Ergebnis:

- Fertige Version von Autix
